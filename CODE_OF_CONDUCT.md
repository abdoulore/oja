# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, and assume good faith. Harassment, personal attacks, and discriminatory behavior are not tolerated anywhere in this project's spaces (issues, pull requests, discussions).

Violations can be reported privately to the maintainer via GitHub. Reports are handled confidentially.
