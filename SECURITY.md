# Security Policy

Ọjà is a hackathon prototype running on Casper **testnet**. No mainnet funds are at risk, but security reports are welcome and taken seriously.

## Reporting a vulnerability

Please report vulnerabilities privately via [GitHub Security Advisories](https://github.com/abdoulore/oja/security/advisories/new). Include steps to reproduce and, if relevant, the contract package hash or service involved.

You can expect an acknowledgement within a few days. Please do not open public issues for security problems.

## Scope notes

- The keys in this repository's examples are for testnet only. Never reuse them for anything of value.
- `.env`, `keys/`, and runtime data are intentionally untracked. If you find any secret material in the repository history, report it as a vulnerability.
